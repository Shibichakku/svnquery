1.0.0.0:  2009-11-21

  • fine tuned syntax highlighting css styles
  • query parameters are now encoded inside the url so you can send links to live query results
  • additional testing for the first 1.0 release

0.8.0.0:  2009-10-26

  • added syntax highlighting fif displaying source code from the repository

0.7.2.0:  2009-10-20

  • Fix: Queries over a range of revisions or an explicit revision could throw a NullReferenceException
  • Fix: Broken Layout on Firefox and IE7

0.7.1.0:  2009-10-19

  • SvnWebQuery: Moved from a table based layout to css based layout. Improves load time and fixes some layout problems with firefox.
  • build process uses 7zip instead of ZipFile task to prevent corrupted zip files
  • resharped the source a little with resharper 4.5

0.7.0.0:  2009-06-15

  • new option -s for generating a "single revision" index, which contains only one revision of an index (without history)
  • much more documentation about how to install and use
  • fixed handling of thread pool exceptions

0.6.0.0:  2009-04-07

  • support for subversion 1.6
  • some documentation
  • cosmetic bugfixes
